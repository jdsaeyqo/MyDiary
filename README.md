# MyDiary

- 시작화면에 비밀번호 세자리를 설정한다.  
- 비밀번호 일치 시 다이어리 액티비티로 이동한다.  
- 다이어리 내용을 로컬에 저장한다.
- 비밀번호 변경 시에는 기존의 비밀번호를 맞춘 상태에서 아래 조그만 버튼을 누른다.  
- 그 후 원하는 번호 설정 후 다시 작은 버튼을 누르면 새롭게 설정된다.

# 사용 기능

- ConstraintLayout, SharedPreference, Handler, AlertDialog
